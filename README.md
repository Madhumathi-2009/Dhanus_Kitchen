# 🍽️ Food Stall App

A digital menu and customer engagement application for our food stall.

The main purpose of this app is to help customers easily view the **daily menu**, available **combos**, **special offers**, communicate with the shop through an **inbuilt chat box**, and provide **feedback**.

The app will initially focus on displaying the menu and building a digital connection with our existing customers.

## 🎯 Project Goal

Currently, many customers contact the shop to ask:

> "What is today's menu?"

This app aims to make that information easily available in one place.

Customers can open the app and immediately view the food available for the day.

## ✨ Features

### 🏠 Today's Menu

* Display the menu for the current day
* Update the menu daily
* Display food items with names, prices, and images
* Mark items as **Sold Out**
* Hide unavailable items when necessary

### 🍱 Combos

Display available food combinations and combo offers.
Example 
White Rice + Fish Gravy

### 🎁 Offers

Show special offers and promotions.

### 💬 Inbuilt Chat Box

Customers can communicate directly with the shop through an inbuilt chat box.

The chat box can be used for:

* Bulk order enquiries
* Functions and party orders
* Special customer requirements
* Food-related enquiries
* General communication with the shop

### ⭐ Feedback

Allow customers to share:

* Feedback
* Suggestions
* Reviews

### 🚚 Future Delivery

Delivery will be introduced in a future version after the business and delivery system are ready.

The app should be designed so that the delivery feature can be enabled later without rebuilding the entire application.

## 🚀 Future Features
